# Frontend Administrativo (Panel React)

Este es el panel de control para administradores y despachadores de Logística Andina. 

## Requisitos Previos
- Node.js (v16 o superior)
- NPM

## Instrucciones de Instalación y Ejecución
1. Abrir una terminal en esta carpeta (`frontend-service`).
2. Instalar las dependencias:
   ```bash
   npm install
   ```
3. Ejecutar el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. El panel estará disponible en `http://localhost:5173`.

*(Nota: Asegúrese de que el Backend `bff-service` esté corriendo en el puerto 8082, o inicie todo el entorno desde la raíz usando `docker-compose up`).*
